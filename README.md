Amazon Sales Analytics Dashboard
📌 Project Overview

This project is a comprehensive Power BI Business Intelligence Solution built using the Amazon Sales Dataset. The dashboard provides end-to-end analysis of sales performance, profitability, customer behavior, product performance, and regional trends, enabling stakeholders to make data-driven business decisions.

The solution transforms raw transactional data into actionable insights through interactive visualizations, KPI tracking, trend analysis, and drill-through reporting.

🎯 Business Objectives
Monitor overall business performance.
Analyze sales trends across time periods.
Identify profitable and loss-making products.
Understand customer purchasing behavior.
Evaluate product and category performance.
Compare regional sales performance.
Support strategic decision-making through interactive dashboards.
